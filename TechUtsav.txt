Project Vision : Secure login with Api GateWay Intigration in the applicaion using (IDS) Identity System server.
Problem Statement : For Multiple Application login Credential details saved in different database server with defferent access rights.
Solution : For above problem statement IDS Api GateWay provides the solution by giving collabrated and identical credential rights with single secured Api Gateway.
